# Ring Portal (Django)

Django version of the Ring + 100G Utilization Dashboard.

## Quick start

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

Open:
- http://127.0.0.1:8000/


This version uses a full-width layout for tables and tabs.
